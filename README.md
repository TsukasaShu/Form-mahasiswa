# Laravel

Repositori ini dibuat untuk pengumpulan tugas menggunakan laravel



## Instalasi

1. Clone repositori 
    ```bash
    git clone https://github.com/TsukasaShu/Form-mahasiswa.git
    ```
2. Pindah ke repositori yang sudah di clone
    ```bash
    cd Form-mahasiswa
    ```
3. Install dependensi laravel:
    ```bash
    composer install
    ```
4. Siapkan file env
    ```bash
    cp .env.example .env
    ```
5. Buat generate application key:
    ```bash
    php artisan key:generate
    ```
6. Jalankan server:
    ```bash
    php artisan serve
    ```

## Teknologi Yang Digunakan

- Laravel Framework
- PHP
